# 28-1st-Gamsung-backend

> [Samsung Electronics](https://www.samsung.com/sec/) is a multinational electronics manufacturer.
> To practice implementing e-commerce website features, we selected one product line (mobile — smartphones) for this project.

</br>

## Team & timeline
- Period: 2021-12-27 ~ 2022-01-07
- Front-end: https://github.com/wecode-bootcamp-korea/28-1st-Gamsung-frontend
- Back-end: Minsung Kang, Byungyeon Jung
  - Shared: ERD/modeling, CSV uploader
  - Minsung Kang: ProductListView, ProductDetailView
  - Byungyeon Jung: SignUpView, SignInView, CartView, login decorator, validation checks

## Collaboration tools
- Slack
- GitHub
- Trello

## Tech stack
- Python, Django, MySQL, Git

## Library
- JWT

## Features

### User
- Sign up: registration with validation checks
- Decorator: issues a JWT token on login and registers the host via token authorization
- Cart: cart CRUD

### Product
- Filtering, sorting, search
- Pagination
